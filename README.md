
# Gerador de Esquemas Cromáticos
Este projeto em Python gera diferentes esquemas cromáticos (complementares, análogos, triádicos e tetrádicos) a partir de uma cor escolhida pelo usuário.

A interface foi construída com Tkinter, permitindo selecionar qualquer cor por meio de um color picker interativo. O programa então calcula as combinações usando a biblioteca colorsys e exibe os resultados em blocos coloridos, de forma simples e visual.
